# Simon Says Game 🎮

## About the Project:-
Simon Says is a classic memory game where the player has to repeat the sequence of colors generated by the game. With every level, the sequence becomes longer, making the game more challenging.This project is built using basic front-end technologies and demonstrates the use of DOM manipulation, event handling, and game logic in JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation & Event Handling)

- ## Features
- Random color sequence generation
- Increasing difficulty level
- Interactive UI with color buttons
- Keyboard event to start the game
- Game over detection
- Score / Level display

- ## How to Play
1. Press any key to start the game.
2. The game will flash a color.
3. Remember the sequence and click the colors in the same order.
4. Each level adds a new color to the sequence.
5. If you click the wrong color, the game ends.
